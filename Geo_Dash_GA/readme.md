# Geo Dash Genetic Algorithm

Genetic Algorithms (GA) are powerful tools for solving problems that don't have a measurable score throughout the process. The video game Geo Dash does a good job as providing a goal ( reach the end ) with vague steps inbetween to accomplishing that goal

[Video over topic](https://youtu.be/2qU_vAnp1vo)

- Deo Dash (a terminal Geometry Dash clone)
